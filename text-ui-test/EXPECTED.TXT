 ITEMTASKER — COMMAND REFERENCE
__________________________________________________________________________
 SKU MANAGEMENT
   addsku n/SKU_ID l/LOCATION          Add a SKU to the warehouse.
   deletesku n/SKU_ID                  Remove a SKU and all its tasks.

 TASK MANAGEMENT
   addskutask n/SKU_ID d/DUE_DATE      Add a HIGH priority task (default).
         [p/PRIORITY]                  Optional: HIGH, MEDIUM, or LOW.
   deletetask n/SKU_ID i/TASK_INDEX    Delete task at given index.
   marktask n/SKU_ID i/TASK_INDEX      Mark a task as completed.
   unmarktask n/SKU_ID i/TASK_INDEX    Unmark a completed task.

 VIEWING
   listtasks n/SKU_ID                  List tasks for a specific SKU.
   listtasks p/PRIORITY                List tasks filtered by priority.
   listtasks l/LOCATION                List tasks sorted by distance.
   listtasks                           List all tasks.

 OTHER
   help                                Show this help message.
   bye / exit                          Exit ItemTasker.
