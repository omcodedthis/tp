 ItemTasker - CLI SKU Ticketing System
 Type a command to get started. Type 'help' for a list of commands.
____________________________________________________________________________________________________
> ____________________________________________________________________________________________________
 ITEMTASKER - COMMAND REFERENCE
____________________________________________________________________________________________________
 SKU MANAGEMENT
   addsku n/SKU_ID l/LOC                                  Add a SKU to the warehouse.
   editsku n/SKU_ID l/NEW_LOC                             Move a SKU to a new location.
   deletesku n/SKU_ID                                     Remove a SKU and all its tasks.

 TASK MANAGEMENT
   addskutask n/SKU_ID d/DUE_DATE [p/PRIO] [t/DESC]       Add a task (priority/desc optional).
   edittask n/SKU_ID i/IDX [d/DATE] [p/PRIO] [t/DESC]     Edit a task (at least 1 field req).
   deletetask n/SKU_ID i/IDX                              Delete task at given index.
   marktask n/SKU_ID i/IDX                                Mark a task as completed.
   unmarktask n/SKU_ID i/IDX                              Unmark a completed task.
   sorttasks n/SKU_ID s/SORT_BY [o/ORDER]                 Sort tasks by date|priority|status.

 VIEWING
   listtasks                                              List all tasks.
   listtasks n/SKU_ID                                     List tasks for a specific SKU.
   listtasks p/PRIORITY                                   List tasks filtered by priority.
   listtasks l/LOCATION                                   List tasks sorted by distance.
   find [n/SKU_ID] [t/DESC] [i/IDX]                       Search tasks by SKU, description, index.
   viewmap                                                Show warehouse map.
   status [n/SKU_ID]                                      Show completion status for SKU(s).

 OTHER
   export                                                 Export inventory to a text file.
   help                                                   Show this help message.
   bye / exit                                             Exit ItemTasker.
